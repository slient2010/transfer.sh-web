使用技巧

如何上传文件

# Uploading is easy using curl $ curl --upload-file ./hello.txt https://fs.teyixing.com/hello.txt https://fs.teyixing.com/66nb8/hello.txt $ curl -H "Max-Downloads: 1" -H "Max-Days: 5" --upload-file ./hello.txt https://fs.teyixing.com/hello.txt https://fs.teyixing.com/66nb8/hello.txt # Download the file $ curl https://fs.teyixing.com/66nb8/hello.txt -o hello.txt

增加shell function 到.bashrc 或者 .zshrc

# Add this to .bashrc or .zshrc or its equivalent transfer(){ if [ $# -eq 0 ];then echo "No arguments specified.\nUsage:\n transfer <file|directory>\n ... | transfer <file_name>">&2;return 1;fi;if tty -s;then file="$1";file_name=$(basename "$file");if [ ! -e "$file" ];then echo "$file: No such file or directory">&2;return 1;fi;if [ -d "$file" ];then file_name="$file_name.zip" ,;(cd "$file"&&zip -r -q - .)|curl --progress-bar --upload-file "-" "https://fs.teyixing.com/$file_name"|tee /dev/null,;;else cat "$file"|curl --progress-bar --upload-file "-" "https://fs.teyixing.com/$file_name"|tee /dev/null;fi;else file_name=$1;curl --progress-bar --upload-file "-" "https://fs.teyixing.com/$file_name"|tee /dev/null;fi;} # Now you can use transfer function $ transfer hello.txt

批量上传文件

$ curl -i -F filedata=@/tmp/hello.txt -F filedata=@/tmp/hello2.txt https://fs.teyixing.com/ # Combining downloads as zip or tar archive $ curl https://fs.teyixing.com/(15HKz/hello.txt,15HKz/hello.txt).tar.gz $ curl https://fs.teyixing.com/(15HKz/hello.txt,15HKz/hello.txt).zip

上传之前加密文件

# Encrypt files with password using gpg $ cat /tmp/hello.txt|gpg -ac -o-|curl -X PUT --upload-file "-" https://fs.teyixing.com/test.txt # Download and decrypt $ curl https://fs.teyixing.com/1lDau/test.txt|gpg -o- > /tmp/hello.txt

# Scan for malware or viruses using Clamav $ wget http://www.eicar.org/download/eicar.com $ curl -X PUT --upload-file ./eicar.com https://fs.teyixing.com/eicar.com/scan # Upload malware to VirusTotal, get a permalink in return $ curl -X PUT --upload-file nhgbhhj https://fs.teyixing.com/test.txt/virustotal

备份数据库, 加密再上传

# Backup, encrypt and transfer $ mysqldump --all-databases|gzip|gpg -ac -o-|curl -X PUT --upload-file "-" https://fs.teyixing.com/test.txt

发送带分享连接的邮件(uses shell function)

# Transfer and send email with link (uses shell function) $ transfer /tmp/hello.txt | mail -s "Hello World" user@yourmaildomain.com

Using Keybase.io

# Import keys from keybase $ keybase track [them] # Encrypt for recipient(s) $ cat somebackupfile.tar.gz | keybase encrypt [them] | curl --upload-file '-' https://fs.teyixing.com/test.txt # Decrypt $ curl https://fs.teyixing.com/sqUFi/test.md |keybase decrypt
